HOW TO GET USER posts USING ACCESS TOKEN

1. LOGIN
2. COPY ACCESS TOKEN
3. PASTE ACCESS TOKEN TO GET /posts REQUEST
3. GET /posts

HOW TO GET USER posts USING REFRESH TOKEN

1. LOGIN
2. COPY REFRESH TOKEN
3. PASTE REFRESH TOKEN TO POST /token REQUEST
4. POST /token
5. COPY ACCESS TOKEN
6. PASTE ACCESS TOKEN TO GET /posts REQUEST
7. GET /posts

HOW TO GET USER posts USING REFRESH TOKEN WITHOUT LOGIN
4. POST /token (using same refresh token given at login)
5. COPY ACCESS TOKEN
6. PASTE ACCESS TOKEN TO GET /posts REQUEST
7. GET /posts

HOW TO LOGOUT
1. PASTE REFRESH TOKEN given at login TO DELETE /logout REQUEST
2. DELETE /logout
